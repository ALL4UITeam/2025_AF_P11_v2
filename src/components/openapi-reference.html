소개

  • 국립해양조사원 개방海에서는 海아름 해양배경지도와 다양한 해양공간 정보를 제공합니다.
  • 발급받은 오픈API 인증키와 함께 개방海 오픈API 서비스 URL로 요청 시 WMS / WFS 서비스를 이용 할 수 있습니다.
  • 개방海 오픈API 서비스는 오픈 소스 기반의 OpenLayers( v1 : 2.7~2.13, v2 : 3.x 이상) 사용을 지원합니다.

요청URL

요청주소

  • v1) http://www.khoa.go.kr/oceanmap/<베이스맵 타입>/otmsVectormapApi.do?ServiceKey=<인증키>
  • v2) http://www.khoa.go.kr/oceanmap/<베이스맵 타입>/otmsVectormapApi.do?ServiceKey=<인증키>&version=2
  • v3) https://www.khoa.go.kr/oceanmap/<베이스맵 타입>/otmsSSLVectormapApi.do?ServiceKey=<인증키> (* https 요청 시)

사용예

  • <script src="http://www.khoa.go.kr/oceanmap/BASEMAP_ENC57/otmsVectormapApi.do?ServiceKey=<인증키>&version=2"></script>
  • <script src="https://www.khoa.go.kr/oceanmap/BASEMAP_ENC57/otmsSSLVectormapApi.do?ServiceKey=<인증키>&version=2"></script> (* https 요청 시)

※실시간 타일방식의 지도 호출방식은 BBOX,XYZ방식으로 나누어져 있으므로 사용에 유의하시길 바랍니다. (상세한내용은 [海아름 베이스맵 사용예제] 해아름 사용예제 다운로드참고)

WMS 방식

요청주소

  • v1) http://www.khoa.go.kr/oceanmap/otmsWmsApi.do?ServiceKey=<인증키>&Layer=<레이어명>
  • v2) http://www.khoa.go.kr/oceanmap/otmsWmsApi.do?ServiceKey=<인증키>&Layer=<레이어명>&version=2

좌표계

  • 좌표계 : UTM-K, EPSG:5179

스타일명

  • 스타일명은 Open-API제공 목록을 참조해 주십시오.

WFS 방식

요청주소

  • v1,v2 동일) http://www.khoa.go.kr/oceanmap/otmsWfsApi.do?ServiceKey=<인증키>&Layer=<레이어명>&FILTER=<필터조건>&minlon=<좌하단최소경도>&minlat=<좌하단최소위도>&maxlon=<우상단최대경도>&maxlat=<우상단최대위도>[&srsName=EPSG:5179]

srsName 기본값

  • EPSG:5179

데이터) XML / JSON 방식

요청주소

  • http://www.khoa.go.kr/oceanmap/otmsInfoApi.do?ServiceKey=<인증키>&Layer=<레이어명>&numOfRows=<페이지결과수>&resultType=<파일형태>&pageNo=<페이지번호>&SIDOCD=<시도코드>&SIGUNGUCD=<시군구코드>
    ※데이터별 요청 파라미터는 하단 [OpenAPI제공 목록]파일 다운로드 후 확인
    ※numOfRows, pageNo, resultType,SIDOCD,SIGUNGUCD 는 선택사항으로 필수 입력사항은 아닙니다. [기본값: numOfRows(10), pageNo:(1),resultType:(xml),SIDOCD:(전국)]

사용예

  • XML형식) http://www.khoa.go.kr/oceanmap/otmsInfoApi.do?ServiceKey=<인증키>&Layer=<레이어명>&numOfRows=<페이지결과수>&SIDOCD=<시도코드>&resultType=xml
  • JSON형식) http://www.khoa.go.kr/oceanmap/otmsInfoApi.do?ServiceKey=<인증키>&Layer=<레이어명>&numOfRows=<페이지결과수>&SIDOCD=<시도코드>&resultType=json
